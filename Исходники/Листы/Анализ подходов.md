# Подходы к извлечению информации из текста
1. На основе кластеризации патентов
2. На основе ресурсных сетей
3. На основе машинного обучения

## 1. Подход на основе кластеризации патентов:
Этот метод включает построение иерархической кластерной структуры (дендрограммы) патентов для разных моментов времени, где каждый кластер соответствует определенной технологической области. Сопоставление дендрограмм помогает обнаружить эволюцию кластеров, такие как их появление, исчезновение, рост, уменьшение, объединение и разделение.

Пример использования: кластеризация на основе вектора цитирования  $v_i=(c_1, c_2, ..., c_k)$, где $c_k$ — количество патентов, относящихся к $k$-ому патентному классу.

**Преимущества:**
 - Позволяет обнаруживать новые технологические области через анализ эволюции кластеров.
 - Обеспечивает визуализацию структуры патентов в виде дендрограммы.


**Недостатки:**
 - Высокая вычислительная сложность, особенно при обновлении данных.
 - Сложность установления соответствий между кластерами в различных временных точках.
 - Трудоемкость анализа дендрограмм экспертом.

## 2. Подход на основе ресурсных сетей:
Ресурсные сети представляют собой динамические графовые модели распространения ресурса. Вершины графа могут хранить ресурс (например, патенты), а ребра имеют ограниченную пропускную способность. Процесс прогнозирования основывается на анализе движения ресурса по сети с учетом вероятностных моделей, таких как цепи Маркова.

Пример: Стохастическая матрица переходов $P$, где $P_{ij}$ — вероятность перехода из состояния $i$ в состояние $j$.

**Преимущества:**
 - Математически обоснованный подход с использованием теории вероятностей.
 - Позволяет учитывать различные виды связей между патентами и объектами классификатора.
 - Возможность моделирования распространения ресурсов и прогнозирования появления новых объектов на основе анализа текущих сетей.

**Недостатки:**
 - Требуется добавление дополнительных узлов-стоков для прогнозирования.
 - Определение количества и связей новых узлов может быть сложной задачей.
 - Требует точной настройки модели для каждой специфической задачи.


## 3. Подход на основе машинного обучения:
Методы машинного обучения, такие как регрессия и классификация, используются для предсказания появления новых объектов классификатора. Каждый объект представляется вектором признаков, и прогнозирование выполняется на основе анализа этих признаков.

Пример: Логистическая регрессия для классификации, где $P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 x_1 + ... + \beta_n x_n)}}$, где $y$ — целевая переменная, $x$ — вектор признаков, $\beta$ — параметры модели.

Пример векторизации: модели BERT, RoBERTa, GPT-2, обученные на миллионах текстовых документов.

**Преимущества:**
 - Низкая вычислительная сложность анализа новых данных.
 - Высокая интерпретируемость результатов прогнозирования.
 - Возможность автоматического определения признаков с высокой предсказательной силой.

**Недостатки:**
 - Требуется подготовка обучающей выборки на основе исторических данных.
 - Необходимость регулярного обновления моделей на основе новых данных.
 - Влияние качества входных данных на точность прогнозирования.

## Результаты анализа
Анализ существующих подходов показал, что каждый из них имеет свои сильные и слабые стороны. Подход на основе кластеризации обладает высокой вычислительной сложностью и трудоемкостью анализа результатов. Подход на основе ресурсных сетей требует значительной доработки и оптимизации, однако обладает потенциалом для прогнозирования на основе различных видов связей. Подход на основе машинного обучения лидирует из-за низкой вычислительной сложностью и высокой интерпретируемостью.

**Факторы:**
 - Наличие большого числа данных о патентах
 - Необходимость работы с большими объемами данных.
 - Требования к интерпретируемости результатов.
 - Наличие специалистов в области патентоведения.
 - Наличие вычислительных ресурсов для обработки данных.

В качестве основного подхода к прогнозированию новых технологических областей предлагается использовать синергию рассмотренных методов. Это включает использование предобученных на цитировании моделей для векторизации текстов патентов, кластеризацию для выявления эволюции технологических областей и методы машинного обучения для точного прогнозирования.
